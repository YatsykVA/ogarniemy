# Как самому менять текст сайта

Главный файл с текстами:

`C:\Users\Viktor\Documents\Codex\2026-05-17\new-chat\presentation-site\index.html`

## Самый простой способ

1. Открой файл `index.html` в Блокноте, VS Code или любом редакторе.
2. Найди нужный текст через `Ctrl+F`.
3. Меняй только слова между HTML-тегами.
4. Сохрани файл через `Ctrl+S`.
5. Обнови сайт в браузере через `Ctrl+R`.

## Пример

Было:

```html
Помогаю быстро упаковать идею в красивый сайт-презентацию,
понятный текст и простую автоматизацию с ИИ.
```

Можно заменить на:

```html
Создаю красивые сайты, презентации и визуальные материалы
для личных проектов и малого бизнеса.
```

## Что лучше не ломать

Не удаляй символы `<`, `>`, `</...>`, кавычки внутри ссылок и строки вида `class="..."`.

Например, здесь меняй только слово `Услуги`, но не трогай `<a href="#idea">`:

```html
<a href="#idea">Услуги</a>
```

## Где лежат основные тексты

- Название вкладки браузера: строка с `<title>`.
- Верхнее меню: строки с `Старт`, `Услуги`, `Процесс`, `Пакеты`, `Контакт`.
- Главный текст первого экрана: строки под картинкой `Viktor Yatsyk`.
- Карточки `Сайт`, `Текст`, `ИИ`: блоки внутри `hero-stats`.
- Остальные крупные заголовки: теги `<h2>...</h2>`.

## Важно про имя на первом экране

Сейчас крупная надпись `Viktor Yatsyk` на первом экране сделана картинкой:

```html
<img class="hero-wordmark" src="./assets/viktor-yatsyk-wordmark-en-transparent.png" ...>
```

Если нужно изменить именно эту красивую надпись, надо создать новую PNG-картинку или попросить Codex заменить ее.
