CollectorHub Update32 - Facebook group search/save fix

Что изменено:
1. Убраны лимиты поиска Facebook-групп.
   Больше нет max_per_query, [:12], [:48] и других ограничений по количеству групп.
   Скрипт прокручивает результаты и собирает все группы, которые Facebook подгружает.

2. Название группы теперь сохраняется именно как название группы.
   ID группы не записывается в поле name/title.
   ID хранится только технически в group_id/url.

3. Дубли больше не создаются.
   Если группа уже есть по URL или group_id, запись обновляется, а не добавляется второй раз.

4. При запуске GroupsManager очищает уже появившиеся дубли по group_id/url.

Изменённые файлы:
- facebook_group_search.py
- groups_manager.py
- main.py
