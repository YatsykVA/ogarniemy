# CollectorHub --- Пакет №1

Этот документ описывает первый пакет реализации проекта.

## Структура

-   main.py
-   config.py
-   database/database.py
-   gui/main_window.py
-   gui/dashboard.py
-   facebook/session.py
-   telegram/sender.py

## Реализуемые функции

1.  Запуск Windows GUI на PySide6.
2.  Создание SQLite базы.
3.  Загрузка списка Facebook-групп.
4.  Настройки Telegram.
5.  Логирование.
6.  Подготовка Facebook-сессии.
7.  Каркас Web Collector.

## Следующие файлы

Во втором пакете будут: - collector.py - parser.py - extractor.py -
filters - AI manager - интеграция Telegram.

Это не заглушка проекта, а план первого пакета. Далее файлы будут
содержать полноценный код.
