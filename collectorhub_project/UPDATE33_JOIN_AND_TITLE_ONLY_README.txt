CollectorHub Update 33 - JOIN + TITLE ONLY FIX

Исправлено в автопоиске Facebook-групп:

1. Фильтрация только по названию группы.
   Программа НЕ использует описание, публикации, комментарии или раздел "О группе".
   Берётся только название группы + ссылка.

2. В поле названия сохраняется название группы, не ID.
   ID/URL остаются только техническими полями.

3. Группа сохраняется только после подтверждённого вступления.
   Алгоритм:
   - открыть группу;
   - нажать "Присоединиться к группе" / Join group / Dołącz do grupy;
   - подождать примерно 5 секунд;
   - проверить, что Facebook подтвердил вступление или что пользователь уже участник;
   - только после этого добавить/обновить группу в списке.

4. Если нужна заявка, вопросы или вступление не подтверждено — группа НЕ сохраняется.

5. Лимита на количество найденных групп нет.
   Остановка происходит только когда Facebook перестаёт подгружать новые результаты.
