Update 36 - Facebook Share Arrow Fix

Исправление по требованию:
- НЕ создаём новый пост в целевой группе.
- Используем оригинальную кнопку Facebook со стрелочкой: Поделиться / Share / Udostępnij.
- Открываем оригинальное объявление по post_url.
- Нажимаем Поделиться.
- Выбираем режим "Поделиться в группе".
- Выбираем целевую группу из списка CollectorHub.
- Нажимаем "Поделиться сейчас".
- Пост помечается отправленным в Facebook только после успешного нажатия финальной кнопки.

Важно:
- Если у объявления нет post_url, его нельзя честно переслать стрелочкой. В таком случае программа пишет ошибку в лог и не создаёт копию руками.
- Целевая группа должна быть выбрана в выпадающем списке Facebook-группы-получателя.
