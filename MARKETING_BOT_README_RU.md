# Маркетинговый бот Ogarniemy

Это первая рабочая основа для продвижения через Telegram и Facebook.

Что умеет:

- принимать подписчиков в Telegram через `/start`;
- разделять людей на клиентов `/client` и мастеров `/worker`;
- сохранять город через `/city Warszawa`;
- рассылать сообщения только тем, кто сам подписался;
- добавлять Telegram-группы для публикаций;
- смотреть ключевые слова в добавленных группах;
- публично отвечать в группе, если найдено ключевое слово;
- ограничивать частоту ответов по ключевым словам, чтобы группу не забанили;
- принимать входящие сообщения Facebook Messenger через webhook.

Важно: бот не отправляет личные сообщения незнакомым людям. Это специально, чтобы не получить блокировку Telegram/Facebook и не испортить репутацию проекта. Для продвижения используем группы, каналы, рекламу и подписчиков, которые сами нажали `/start`.

## Запуск Telegram

1. Создать бота через `@BotFather`.
2. Получить токен.
3. Узнать свой Telegram ID, например через `@userinfobot`.
4. Запустить:

```powershell
$env:TELEGRAM_BOT_TOKEN="токен_бота"
$env:ADMIN_TELEGRAM_IDS="ваш_telegram_id"
$env:PRESENTATION_URL="https://ogarniemy.pro"
python marketing_bot.py --telegram
```

## Команды для пользователей

```text
/start
/client
/worker
/city Warszawa
/stop
```

## Команды администратора

Рассылка подписчикам:

```text
/broadcast all текст
/broadcast clients текст
/broadcast workers текст
```

Публикация во все добавленные группы:

```text
/postgroups текст
```

Статистика:

```text
/stats
```

Список групп:

```text
/groups
```

## Как добавить группу

1. Добавить Telegram-бота в группу.
2. Сделать его администратором, если группа запрещает сообщения обычным участникам.
3. Написать в группе команду:

```text
/watch Warszawa | мастер, сантехник, электрик, ремонт, уборка, переезд
```

После этого бот будет смотреть эти слова в группе. Если кто-то пишет сообщение с ключевым словом, бот публично отвечает рекламным текстом Ogarniemy. Частота ответа ограничена переменной `KEYWORD_REPLY_COOLDOWN`, по умолчанию один раз в 6 часов на одно слово в одной группе.

## Facebook Messenger

Для Facebook нужен Page Access Token и webhook в настройках Meta Developers.

Запуск webhook:

```powershell
$env:FACEBOOK_PAGE_ACCESS_TOKEN="токен_страницы"
$env:FACEBOOK_VERIFY_TOKEN="любой_секретный_текст"
$env:PRESENTATION_URL="https://ogarniemy.pro"
python marketing_bot.py --facebook-webhook --port 8080
```

Webhook URL:

```text
https://ваш-сервер/facebook/webhook
```

Facebook-бот отвечает только тем, кто сам написал в Messenger страницы.

## Что можно улучшить следующим шагом

- добавить расписание автопостов по дням недели;
- сделать отдельные тексты для каждого города;
- сделать категории мастеров: сантехник, электрик, уборка, ремонт;
- добавить QR-код и картинку баннера в посты;
- подключить бота к внешнему серверу Railway отдельным сервисом;
- сделать простую админ-страницу для управления текстами, городами и группами.
