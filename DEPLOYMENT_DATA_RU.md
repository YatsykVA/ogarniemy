# Постоянная база и редеплой

Сервер хранит живые данные отдельно от кода. При редеплое можно обновлять репозиторий, а база, загруженные рекламные картинки, Telegram-сессия и настройки бота останутся в постоянной папке.

## Что поставить на сервере

Создайте папку вне репозитория, например:

```bash
mkdir -p /var/lib/ogarniemy
```

В переменных окружения сервера укажите:

```bash
OGARNIEMY_DATA_DIR=/var/lib/ogarniemy
TASK_ADMIN_PASSWORD=ваш_пароль_админа
```

Обычно этого достаточно. База будет здесь:

```text
/var/lib/ogarniemy/server.db
```

Если нужен точный путь к базе:

```bash
TASK_DB_PATH=/var/lib/ogarniemy/server.db
```

## Что теперь хранится в базе

- логины и пароли сотрудников;
- логины и пароли клиентов;
- задания;
- балансы/кредиты клиентов и сотрудников;
- расчеты;
- группы Telegram;
- группы Facebook;
- рекламные материалы;
- расписание рекламы;
- найденные Facebook-объявления из CollectorHub;
- журналы действий.

## Импорт старого CollectorHub

Для переноса групп и найденных объявлений из старого проекта:

```bash
python import_collectorhub_data.py --project "C:\Users\Viktor\Desktop\Проект"
```

На сервере путь будет другим, например:

```bash
python import_collectorhub_data.py --project /path/to/old/collectorhub
```

После импорта раздел `/facebook-ads` берет группы и найденные объявления уже из постоянной базы.
